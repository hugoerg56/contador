CONTADOR

We tell you how many times a word appears in a string.

Install:

gem install contador

Usage:

require 'contador'

@sentence = Contador.new("Esto sale todo")
=> #<Contador:0x10212a860 @final={:esto=>1, :sale=>1, :todo=>1}>

@sentence.multiplicity
=> {:esto=>1, :sale=>1, :todo=>1}

Omitting Frequent Words:

$omits = "esto no sale"
=> "esto no sale"

@sentence = Contador.new("Esto sale todo")
=> #<Contador:0x102110b68 @final={:todo=>1}>

@sentence.multiplicity
=> {:todo=>1}
